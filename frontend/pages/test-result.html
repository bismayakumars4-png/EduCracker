Q1 Algebra - Quadratic Equations

Given: x² + 5x + 6 = 0

Solution:

Factorizing: (x + 2)(x + 3) = 0

So x = -2 or x = -3

Taking x = -2:

x² + 1/x² = 4 + 1/4 = 4 + 0.25 = 4.25

Wait, let me recalculate using the identity:

(x + 1/x)² = x² + 2 + 1/x²

From the quadratic: x + 2 + 3/x = 0

This gives x + 1/x = -5/2

Then x² + 1/x² = (x + 1/x)² - 2 = 25/4 - 2 = 25/4 - 8/4 = 17/4 = 4.25
